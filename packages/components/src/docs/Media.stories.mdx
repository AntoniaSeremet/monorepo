# Screen Sizes

The component library understands several screen dimensions:
- **desktop** : >= 1200px
- **tabletLandscape** : >= 800px && <1200px
- **tabletPortrait** : >= 600px && <800px
- **mobile** : >= 320px && <600px

These are available inside of storybook and exported as scss variables.
In addition, you can use the `mediaQuery` mixin from the component library to
write simple CSS media queries. Note that it can accept either the before mentioned 
keys or an actual size in pixels.


```scss
// import our main scss before so it's accessable

.exampleClass {
  @include mediaQuery(">=350px", "tabletLandscape") {
    color: blue;
  }

  @include mediaQuery(">desktop") {
    color: red;
  }
}

```

**Note** that our maximum screen size is **1800px** - after that point you should see 
no changes on our components.
