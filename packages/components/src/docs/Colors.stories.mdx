# Colors

This page demonstrates all available colors in this component library. Each color is uniquely defined by
a string name inside of our system and you can pass these string values to specific properties of components.
As an example, you can pass `blue_6` as the value for the `color` property of the `Typography` component.
This will set the text color to "#F6FCFD". 

If you click on the squares of colors below, you'll copy the corresponding color's value of this unique string.
<br/>

### Color Definitions
Inside of our system, we specify all colors that can be used in varios contexts of the application. This is
what we would call a color definition - we do not specify the context for each color (is it a paragraph color or a 
button background?) but simply define it as existing. 

<div class="subheading">All available colors</div>

<div style={{ display: 'flex', flexWrap: 'wrap'}}>
  {
    Object.keys(styles).map(colorKey => {
      if (
        colorKey.startsWith('blue')
        || colorKey.startsWith('gray')
        || colorKey.startsWith('yellow')
        || colorKey.startsWith('red')
        || colorKey.startsWith('green')
        || colorKey.startsWith('orange')
        || colorKey.startsWith('white')
      ) {
        return (
          <button onClick={() => {
            if (navigator && navigator.clipboard && navigator.clipboard.writeText) {
              navigator.clipboard.writeText(colorKey);
            }
          }} key={colorKey} className="color-display">
            <div class="color-background" style={{backgroundColor: styles[colorKey]}}></div>
            <b>{colorKey}</b> <br/>
            {styles[colorKey]}
          </button>
          
        )
      }
      return '';
    })
  }
</div>
<br/>

### Color Decisions
Once we decide a color has usage in some context, we create a contextual variable pointing to a color definition (see
paragraph before). This is what we call a color decision - we decided that a specific color would be used for
a specific purpose for a specific UI element (such as button background color or text color).

<div style={{ display: 'flex', flexWrap: 'wrap'}}>
